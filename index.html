How a text message AI assistant works in practice

1. You text the request

“Reschedule my flight,” “Track this order,” or “Check availability and remind me.” No app switching.

2. Super uses a real computer

Super opens the browser, logs in if needed, and performs the steps — with your confirmation for sensitive actions.

3. Cache reuse

The steps are saved to a computer-use cache, so next time the same task runs faster and more efficiently.

You get updates, clarifying questions, and reminders right back in the conversation.
